# Ogma

> Lector Inteligente de Inglés — lectura EPUB con tap‑to‑translate, offline first.

## 🧠 Visión
Ogma permite leer EPUBs y consultar traducciones al instante **sin salir de la app** y **sin conexión**. Todo es local: libros, progreso, diccionario y vocabulario.

## ✅ MVP (alcance actual)
- Biblioteca local con importación de EPUB
- Lector EPUB con tap‑to‑translate
- Diccionario embebido (JSON/SQLite)
- Persistencia del progreso de lectura

## 🛠 Stack
- React Native (Expo) + TypeScript
- Almacenamiento local (SQLite/Hive)
- Motor de TTS nativo

## 🚀 Quick Start
```bash
cd Ogma
npm install
npm run android
npm run ios
```
